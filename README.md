# Database Projects

## The SQL Database
Implements an SQL Database for a real estate agency. Keeps track of residents, apartments and contracts between them. Users can search for residents by last name to generate a report containing all their information. Users can also generate queries to the database, search and edit it through a comprehensive and easy to use user interface.

## The NoSQL Database
Implements an internet cafe database. Allows users to sign up, and later log in with a username or password. Can query the database for and generate reports on various information relating to installed games, computers present and the customers.
